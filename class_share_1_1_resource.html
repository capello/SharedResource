Member Enumeration Documentation

enum Share::Resource::State

Enumerated to define states available for a resource. The safe states are AVAILABLE, PARTIALY_AVAILABLE, GETTING. In those states the resource has been saved somewhere.

Enumerator:
UNKNOWN 

Resource has not be created.

AVAILABLE 

Resource is localy available and complete.

PARTIALY_AVAILABLE 

Resource is partialy available localy.

GETTING 

Resource has been requested localy and download from peers is progressing.

PARTIAL 

Resource is partial, we cannot retreive all parts.

CONFLICT 

The resource has been modify by two users and the modifications need to be merged.

CREATING 

First version of resource is currently in creation (need resource, owner).

MODIFYING 

The resource is in modification.
