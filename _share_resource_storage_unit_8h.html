Detailed Description

This class store a unit of a Ressource.

Author:
Anthony Jaguenaud
Version:
v0.1.0

A unit is a piece of. For example is you have a unit size equal to 64MB, a ressource of 264MB will be separate into 5 units. Each unit will be replicated over the network. This class can exists only if Share::ShareResourceStorage::s_storage_method == Share::StorageMethod::DISK
